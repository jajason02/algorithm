## Best Practice

1. **가독성 우선** — PEP 8 지키기 (클래스: PascalCase / 함수·변수: snake_case / 상수: SNAKE_CASE)
2. **효율성 체크** — 시간복잡도 O(N)을 항상 계산하고 제출
3. **기록의 힘** — 어떻게 풀이를 시작했는지, 왜 이 알고리즘을 선택했는지, AI의 도움을 어떻게 받았는지 주석으로 생각의 흐름 남기기
